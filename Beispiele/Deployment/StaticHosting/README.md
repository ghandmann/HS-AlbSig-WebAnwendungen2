# Static Webhosting Beispiel

Das ist ein sehr einfaches Beispiel für statisches WebHosting mit NGINX.

Der NGINX Server wird dabei als Container gestartet und liefert alle Dateien und Ordner im Verzeichnis `wwwroot` aus.

Nach dem Starten via `./start-nginx-in-docker.sh` läuft der WebServer auf `http://localhost:8000/`
